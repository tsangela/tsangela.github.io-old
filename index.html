PROJECTS

Hungry Cat (Java)

A simple snake-like game featuring a very hungry cat. You play as the cat! Use the arrow keys to move around, but avoid hitting walls. Different foods have different nutritional values. The more food you eat, the faster you run! During gameplay, press the space bar to pause the game. See more...

  • Utilized JFrame and JPanel to produce double-buffered application window
  • Implemented game states to distinguish game progress
  • Implemented separate render classes to increase cohesion
  • Created a discrete probability distribution corresponding to different food types
  • Performed unit testing with JUnit
